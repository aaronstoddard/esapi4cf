Introduction

These tutorials are largely based on the efforts from the ESAPI Swingset Interactive application. The tutorials have been modified to exclude the functioning aspects of the labs so that this could be for documentative purposes only. All example code used in the tutorials will be made available as part of the ESAPI4CF main project.

ESAPI Swingset Credits

The Swingset Interactive application was structured according to the OWASP Application Security Verification Standard (ASVS) requirements.

Each of the 9 chapters consists of multiple labs, each lab requiring the use of ESAPI to secure a part of the application.
